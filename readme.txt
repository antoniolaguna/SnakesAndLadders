Instrucciones:

Paso 1: Ejecutar SnakesAndLadders/Test/bin/Debug/net6.0/Test.exe
Paso 2: Escribir una de las siguientes opciones de prueba (1-9)

1. Juego completo manualmente --> Recreación del juego de manera completa, generando aleatorios y teniendo que pulsar para cada tirada de datos.
2. Juego completo automaticamente --> Empieza la partida y el sistema juega solo hasta que encuentra un ganador, alternando entre dos jugadores.
3. US1_UAT1 --> Prueba solicitada en US1_UAT1
4. US1_UAT2 --> Prueba solicitada en US1_UAT2
5. US1_UAT3 --> Prueba solicitada en US1_UAT3
6. US2_UAT1 --> Prueba solicitada en US2_UAT1
7. US2_UAT2 --> Prueba solicitada en US2_UAT2
8. US3_UAT1 --> Prueba solicitada en US3_UAT1
9. US3_UAT2 --> Prueba solicitada en US3_UAT2


Después de ejecutar la prueba solicitada, la aplicación se cerrará.
